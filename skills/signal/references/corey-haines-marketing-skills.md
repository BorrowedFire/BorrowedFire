# Corey Haines Marketing Skills Catalog

Treat "Corey Haines Marketing Skills" or "Marketing Skills" as the local marketing skill trove installed under `~/.codex/skills`. Open the selected skills' `SKILL.md` files before using them.

## Default Chain For Copy

Use this chain unless the request clearly calls for a narrower path:

1. `product-marketing` if positioning, audience, offer, or voice context is missing or stale.
2. `customer-research` if the request depends on voice-of-customer language, objections, jobs to be done, review mining, or why users buy/churn.
3. Surface/channel specialist from the tables below.
4. `marketing-psychology` only when the angle needs sharper behavioral framing.
5. `copy-editing` for the final pass on clarity, voice, "so what", specificity, proof, CTA, and friction.

## Practical Selection Rules

- If the user asks for copy but the message is strategically wrong, use foundation skills first. Do not polish the wrong premise.
- If the copy depends on buyer language, run `customer-research` or inspect existing VOC/customer docs before writing.
- If the copy is for a conversion surface, pair the channel skill with `cro`, `signup`, `onboarding`, or `paywalls` as appropriate.
- If the user wants variants or performance testing, add `ad-creative` or `ab-testing`.
- If the user asks for "the Marketing Skills take", consider at least one strategy skill and one surface skill before producing copy.
